# Search & Filter Commands

Find and filter data in files or directories.

### `grep`
Search inside files:
```bash
grep "text" file.txt
```




### `find`
Locate files:

Example:
```
find / -name "*.log"

```

### `sort`
Sort lines of text.

Example:
```
sort
```
